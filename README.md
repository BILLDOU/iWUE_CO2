# iWUE_CO2

This repository contains the code necessary to reproduce the results and figures from the main manuscript of Zhan et al., submitted to a peer-reviewed journal.

## Code Overview

- **Main Figures**: Jupyter notebooks for reproducing key figures (`Fig_1_Fig_2.ipynb`, `Fig_3.ipynb`, `Fig_4.ipynb`, `Fig_5.ipynb`).
- **Method Example**: `method_example.ipynb` demonstrates how we isolate the CO₂ effect on iWUE at an example site.
